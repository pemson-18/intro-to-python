# **Intro to Python – Faculty Edition**

### *An introductory course by Pemson Pereira*

Welcome to this beginner-friendly Python program designed specially for faculty members who want to strengthen their computational skills, simplify academic workflows, and bring modern programming practices into their classrooms. The goal of this course is to make Python accessible, practical, and enjoyable—even for educators who may be touching code for the first time.

## **About the Instructor**

**Pemson Pereira** is an IT consultant and web developer with several years of experience working across industries and technologies. His journey spans web development, custom software consulting, and system analysis, with hands-on work in Drupal, PHP, cloud systems, and data-driven applications. Over the years, he has built and enhanced platforms for media, education, healthcare, and enterprise clients. His experience includes projects for MediaCorp, OpenExchangeTV, data-intensive educational portals, and full-stack application development across various organisations. 

He brings a strong academic foundation with a **B.Sc in Computer Science**, and has completed multiple certifications in **Python**, **R**, **HTML5**, **CSS3**, SQL, JavaScript, PHP, and frontend frameworks. His early exposure to programming and continuous learning makes him passionate about helping faculty adopt computational methods with confidence.

## **What This Course Covers**

* Python basics for absolute beginners
* Writing and understanding simple scripts
* Variables, data types, and expressions
* Using Python for everyday academic tasks
* Introduction to NumPy for scientific work
* Visualizing data with Matplotlib
* Hands-on tasks that teachers can reuse in their own classrooms

## **Course Style**

The course keeps things simple, slow-paced, and practical. Each topic includes examples, exercises, and real-world applications that suit mathematics, physics, chemistry, and general science educators.
